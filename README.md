swing-explorer-idea
===================

[SwingExplorer][1] plugin for Intellij IDEA.

A fork from http://swing-explorer.googlecode.com/svn/trunk/swing-explorer-idea/ that includes minor fixes which make it work with the latest version of IDEA (13.x).
